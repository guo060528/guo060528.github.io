# 复变函数完整知识体系提纲

## 第一章 复数与复变函数

### 1.1 复数及其运算
*   **复数的定义与代数形式**
    *   定义：$$z = x + iy$$ ($$x, y \in \mathbb{R}, i^2 = -1$$)
    *   四则运算：
        *   加减：$$(x_1+iy_1) \pm (x_2+iy_2) = (x_1\pm x_2) + i(y_1\pm y_2)$$
        *   乘法：$$(x_1+iy_1)(x_2+iy_2) = (x_1x_2-y_1y_2) + i(x_1y_2+x_2y_1)$$
        *   除法：$$\frac{z_1}{z_2} = \frac{z_1 \bar{z_2}}{|z_2|^2}$$
*   **共轭复数**
    *   公式：$$\bar{z} = x - iy$$，性质：$$\overline{z_1+z_2} = \bar{z_1} + \bar{z_2}$$
*   **模与不等式**
    *   模：$$|z| = \sqrt{x^2+y^2} = \sqrt{z\bar{z}}$$
    *   三角不等式：$$||z_1|-|z_2|| \leq |z_1+z_2| \leq |z_1|+|z_2|$$

### 1.2 复数的几何表示
*   **复平面与三角形式**
    *   三角形式：$$z = r(\cos\theta + i\sin\theta)$$
    *   关系：$$x=r\cos\theta, y=r\sin\theta, r=|z|, \theta = \arg z$$
*   **指数形式 (欧拉公式)**
    *   公式：$$z = re^{i\theta}$$ ($$e^{i\theta} = \cos\theta + i\sin\theta$$)
*   **乘除法的几何意义**
    *   乘法：模相乘，辐角相加 ($$r_1r_2 e^{i(\theta_1+\theta_2)}$$)
    *   除法：模相除，辐角相减
*   **乘方与开方**
    *   棣莫弗公式：$$z^n = r^n (\cos n\theta + i\sin n\theta)$$
    *   开方根：$$w_k = \sqrt[n]{r} e^{i\frac{\theta + 2k\pi}{n}}, \quad k=0,1,\dots,n-1$$

### 1.3 复球面与无穷大
*   **扩充复平面**
    *   定义：$$\mathbb{C} \cup \{\infty\}$$
*   **黎曼球面**
    *   北极点对应 $$\infty$$，南极点对应原点。

### 1.4 复变函数与极限
*   **区域的概念**
    *   邻域：$$|z - z_0| < \delta$$
    *   单连通域与多连通域。
*   **极限与连续**
    *   极限定义：$$\lim_{z\to z_0} f(z) = A \Leftrightarrow \lim_{(x,y)\to(x_0,y_0)} u(x,y)=u_0, v(x,y)=v_0$$
    *   连续性：$$\lim_{z\to z_0} f(z) = f(z_0)$$

## 第二章 解析函数

### 2.1 解析函数的概念
*   **导数的定义**
    *   公式：$$f'(z_0) = \lim_{\Delta z \to 0} \frac{f(z_0+\Delta z) - f(z_0)}{\Delta z}$$
*   **解析的定义**
    *   在点 $$z_0$$ 的邻域内处处可导。

### 2.2 柯西-黎曼方程 (C-R条件)
*   **函数解析的充要条件**
    *   设 $$f(z) = u(x,y) + iv(x,y)$$，则 $$f(z)$$ 可导 $\Leftrightarrow$ $$u, v$$ 可微且满足 C-R 方程：
        $$ \frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x} $$
    *   导数公式：$$f'(z) = \frac{\partial u}{\partial x} + i\frac{\partial v}{\partial x}$$

### 2.3 初等函数
*   **指数函数**
    *   定义：$$e^z = e^x(\cos y + i\sin y)$$
    *   周期性：$$T = 2\pi i$$
*   **对数函数**
    *   定义：$$\operatorname{Ln} z = \ln|z| + i(\arg z + 2k\pi)$$ (多值函数)
    *   主值：$$\ln z = \ln|z| + i \arg z$$ ($$-\pi < \arg z \leq \pi$$)
*   **幂函数**
    *   定义：$$z^a = e^{a \operatorname{Ln} z}$$
*   **三角函数与双曲函数**
    *   $$\sin z = \frac{e^{iz} - e^{-iz}}{2i}, \quad \cos z = \frac{e^{iz} + e^{-iz}}{2}$$
    *   $$\sinh z = \frac{e^z - e^{-z}}{2}, \quad \cosh z = \frac{e^z + e^{-z}}{2}$$

### 2.4 解析函数与调和函数
*   **调和函数**
    *   拉普拉斯方程：$$\Delta u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$
*   **关系**
    *   解析函数的实部和虚部均为调和函数。
    *   已知实部 $$u$$，利用 C-R 方程求虚部 $$v$$ (调和共轭)。

## 第三章 复变函数的积分

### 3.1 积分的定义与性质
*   **定义**
    *   $$\int_C f(z) dz = \int_C (u dx - v dy) + i \int_C (v dx + u dy)$$
*   **估值不等式**
    *   $$|\int_C f(z) dz| \leq M \cdot L$$ （$$M$$ 为 $$|f(z)|$$ 上界，$$L$$ 为曲线长）

### 3.2 柯西积分定理
*   **单连通域柯西定理**
    *   若 $$f(z)$$ 在单连通域 $$D$$ 内解析，则闭路积分 $$\oint_C f(z) dz = 0$$。
*   **复合闭路定理**
    *   多连通域：$$\oint_{C} f(z) dz = \sum_{k=1}^n \oint_{C_k} f(z) dz$$ (外边界积分等于内边界积分之和)。

### 3.3 柯西积分公式
*   **基本公式**
    *   $$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z-z_0} dz$$
*   **高阶导数公式**
    *   $$f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C \frac{f(z)}{(z-z_0)^{n+1}} dz$$
    *   结论：解析函数具有任意阶导数。

## 第四章 级数

### 4.1 复数项级数
*   **收敛性**
    *   $$\sum z_n$$ 收敛 $\Leftrightarrow$ 实部虚部级数同时收敛。

### 4.2 幂级数与泰勒级数
*   **幂级数**
    *   形式：$$\sum_{n=0}^\infty c_n (z-z_0)^n$$
    *   收敛半径：$$R = \frac{1}{\limsup \sqrt[n]{|c_n|}}$$ 或 $$\lim |\frac{c_n}{c_{n+1}}|$$
*   **泰勒展开**
    *   $$f(z) = \sum_{n=0}^\infty \frac{f^{(n)}(z_0)}{n!} (z-z_0)^n$$ (在解析圆内)

### 4.3 洛朗级数 (Laurent Series)
*   **展开定理**
    *   在圆环域 $$R_1 < |z-z_0| < R_2$$ 内：
        $$f(z) = \sum_{n=-\infty}^\infty c_n (z-z_0)^n$$
    *   系数：$$c_n = \frac{1}{2\pi i} \oint_C \frac{f(z)}{(z-z_0)^{n+1}} dz$$

## 第五章 留数及其应用

### 5.1 孤立奇点
*   **可去奇点**
    *   $$\lim_{z\to z_0} f(z)$$ 存在且有限；洛朗级数无负幂项。
*   **极点**
    *   $$\lim_{z\to z_0} f(z) = \infty$$；洛朗级数负幂项有限 ($$m$$ 阶极点)。
*   **本性奇点**
    *   极限不存在；洛朗级数有无穷多负幂项。

### 5.2 留数定理
*   **定义**
    *   $$\operatorname{Res}[f, z_0] = c_{-1} = \frac{1}{2\pi i} \oint_C f(z) dz$$
*   **留数定理**
    *   $$\oint_C f(z) dz = 2\pi i \sum_{k=1}^n \operatorname{Res}[f, z_k]$$

### 5.3 留数的计算
*   **一级极点 (单极点)**
    *   $$\operatorname{Res}[f, z_0] = \lim_{z\to z_0} (z-z_0)f(z)$$
    *   若 $$f(z) = \frac{P(z)}{Q(z)}$$，则 $$\operatorname{Res} = \frac{P(z_0)}{Q'(z_0)}$$
*   **m 级极点**
    *   $$\operatorname{Res}[f, z_0] = \frac{1}{(m-1)!} \lim_{z\to z_0} \frac{d^{m-1}}{dz^{m-1}} [(z-z_0)^m f(z)]$$

### 5.4 留数在定积分计算中的应用
*   **型 I: $$\int_0^{2\pi} R(\cos\theta, \sin\theta) d\theta$$**
    *   代换：$$z=e^{i\theta}, d\theta = \frac{dz}{iz}$$
*   **型 II: $$\int_{-\infty}^{+\infty} R(x) dx$$**
    *   利用上半平面的留数：$$2\pi i \sum \operatorname{Res}$$ (需满足无穷远点收敛条件)。
*   **型 III: $$\int_{-\infty}^{+\infty} R(x) e^{iax} dx$$ (约当引理)**
    *   利用上半平面留数，考虑 $$e^{iaz}$$ 的衰减性。

## 第六章 共形映射

### 6.1 导数的几何意义
*   **伸缩率与旋转角**
    *   $$|f'(z_0)|$$ 表示伸缩率，$$\arg f'(z_0)$$ 表示旋转角。
*   **共形映射概念**
    *   解析且 $$f'(z) \neq 0$$ 的映射具有保角性和伸缩率不变性。

### 6.2 分式线性映射 (Mobius变换)
*   **形式**
    *   $$w = \frac{az+b}{cz+d} \quad (ad-bc \neq 0)$$
*   **性质**
    *   保圆性 (圆或直线映射为圆或直线)。
    *   保对称点性。
*   **唯一确定条件**
    *   三点决定一个分式线性映射：$$\frac{w-w_1}{w-w_2} \cdot \frac{w_3-w_2}{w_3-w_1} = \frac{z-z_1}{z-z_2} \cdot \frac{z_3-z_2}{z_3-z_1}$$

### 6.3 几个典型初等函数的映射
*   **幂函数** $$w=z^n$$：角形域变大/变小。
*   **指数函数** $$w=e^z$$：带状域变为角形域/全平面。
*   **儒可夫斯基变换** $$w = \frac{1}{2}(z + \frac{1}{z})$$：圆域变为椭圆域或线段。

## 第七章 傅里叶变换

### 7.1 傅里叶积分
*   **积分公式**
    *   $$f(t) = \int_{-\infty}^{+\infty} \left[\frac{1}{2\pi} \int_{-\infty}^{+\infty} f(\tau)e^{-j\omega\tau} d\tau\right] e^{j\omega t} d\omega$$

### 7.2 傅里叶变换 (FT)
*   **定义**
    *   正变换：$$F(\omega) = \mathscr{F}[f(t)] = \int_{-\infty}^{+\infty} f(t) e^{-j\omega t} dt$$
    *   反变换：$$f(t) = \mathscr{F}^{-1}[F(\omega)] = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\omega) e^{j\omega t} d\omega$$
*   **单位脉冲函数 $$\delta(t)$$**
    *   筛选性质：$$\int_{-\infty}^{+\infty} \delta(t-t_0) f(t) dt = f(t_0)$$
    *   变换：$$\mathscr{F}[\delta(t)] = 1$$

### 7.3 傅里叶变换的性质
*   **基本性质**
    *   线性、位移特性 ($$F(\omega)e^{-j\omega t_0}$$)、频移特性。
    *   微分性质：$$\mathscr{F}[f'(t)] = j\omega F(\omega)$$
*   **卷积定理**
    *   $$\mathscr{F}[f_1(t) * f_2(t)] = F_1(\omega) \cdot F_2(\omega)$$
    *   卷积定义：$$f_1 * f_2 = \int_{-\infty}^{+\infty} f_1(\tau)f_2(t-\tau) d\tau$$

## 第八章 拉普拉斯变换

### 8.1 拉普拉斯变换的概念
*   **定义**
    *   $$F(s) = \mathscr{L}[f(t)] = \int_0^{+\infty} f(t) e^{-st} dt \quad (s = \sigma + j\omega)$$
*   **存在定理**
    *   $$f(t)$$ 局部可积且增长速度不超过指数级 $$|f(t)| \leq Me^{\beta t}$$。

### 8.2 拉普拉斯变换的性质
*   **微分性质**
    *   $$\mathscr{L}[f'(t)] = sF(s) - f(0)$$
    *   $$\mathscr{L}[f''(t)] = s^2F(s) - sf(0) - f'(0)$$
*   **积分性质**
    *   $$\mathscr{L}[\int_0^t f(\tau)d\tau] = \frac{F(s)}{s}$$
*   **卷积定理**
    *   $$\mathscr{L}[f_1 * f_2] = F_1(s) \cdot F_2(s)$$

### 8.3 拉普拉斯逆变换
*   **反演积分**
    *   $$f(t) = \frac{1}{2\pi i} \int_{\sigma-i\infty}^{\sigma+i\infty} F(s) e^{st} ds$$
*   **留数法求逆变换**
    *   $$f(t) = \sum \operatorname{Res}[F(s)e^{st}, s_k]$$

### 8.4 应用：解常微分方程
*   **步骤**
    1.  对方程两边取拉氏变换 (将微分方程变为代数方程)。
    2.  解出 $$Y(s)$$。
    3.  取逆变换 $$y(t) = \mathscr{L}^{-1}[Y(s)]$$。

## 第九章 应用问题选读 (综合应用)

### 9.1 离散信号处理基础
*   **快速傅里叶变换 (FFT)**
    *   利用对称性和周期性加速 DFT 计算，$$O(N^2) \to O(N\log N)$$。
*   **Z变换**
    *   离散序列的复频域分析：$$X(z) = \sum_{n=-\infty}^{\infty} x[n] z^{-n}$$。

### 9.2 辐角原理与鲁歇定理
*   **辐角原理**
    *   $$\frac{1}{2\pi} \Delta_C \arg f(z) = N - P$$ (零点数 - 极点数)。
*   **鲁歇定理 (Rouche's Theorem)**
    *   若在边界上 $$|f(z)| > |g(z)|$$，则 $$f(z)$$ 与 $$f(z)+g(z)$$ 在内部有相同个数的零点。

### 9.3 平面场的复势
*   **复势函数**
    *   $$W(z) = \phi(x,y) + i\psi(x,y)$$
    *   物理意义：$$\phi$$ 为势函数 (如电势、速度势)，$$\psi$$ 为流函数 (流线、磁感线)。
    *   速度/场强：$$V = \overline{W'(z)}$$。
